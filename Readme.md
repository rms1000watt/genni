# Genni

## Introduction

Generate Go structs using Proto files

## Contents

- [Install](#install)
- [Usage](#usage)

## Install

```bash
go get github.com/rms1000watt/genni
```

## Usage

```
Genni generates go structs from proto files

Usage:
  genni [flags]

Flags:
  -h, --help        help for genni
  -i, --in string   Proto file to read in
```
